Connect to GitHub

To publish content directly to your website, you'll need a GitHub Personal Access Token. This is a one-time setup — your token is stored only in your browser.

  1. Go to GitHub → Settings → Personal Access Tokens
  2. Click "Generate new token"
  3. Give it a name like "NBAG Content Manager"
  4. Under Repository access, select "Only select repositories" and choose your NBAG repo
  5. Under Permissions → Repository permissions, set Contents to "Read and write"
  6. Click "Generate token" and paste it below
