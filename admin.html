One-time setup

Set Up Admin Access

This connects the content manager to your GitHub repository. You only need to do this once.

Step 1 — Get a GitHub token:

  1. Go to GitHub → Personal Access Tokens
  2. Click "Generate new token"
  3. Name it "NBAG Content Manager"
  4. Set expiration to "No expiration" (or your preference)
  5. Under Repository access → "Only select repositories" → choose NBAG
  6. Under Permissions → Contents → set to "Read and write"
  7. Click "Generate token" and copy it
This will be encrypted with your password and stored in your browser.

Step 2 — Choose a password for John:

This is what John will type to sign in. Keep it simple and memorable.
